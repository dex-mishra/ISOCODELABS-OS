Spacing

Spacing is the primary tool for organizing information.

Before typography, color, motion or illustration influence perception, spacing determines how the brain groups, separates and prioritizes information.

Good spacing is largely invisible.

Users should rarely notice spacing directly.

Instead, they should experience an interface that feels calm, structured and effortless to understand.

Whitespace is not unused space.

Whitespace is active communication.

⸻

Spacing Philosophy

Spacing communicates relationships.

Objects positioned close together are perceived as belonging together.

Objects separated by sufficient whitespace are perceived as independent concepts.

Every spacing decision should answer one question:

“What relationship should the user perceive?”

Never add or remove spacing arbitrarily.

✓ Consistent spacing scale

✓ Comfortable component padding

✓ Clear conceptual grouping

✓ Distinct section hierarchy

✓ Appropriate information density

✓ Predictable responsive spacing

✓ Comfortable reading rhythm

✓ Minimal visual clutter
