Media Strategy

Media is not decoration.

Media is evidence.

Every image, video, illustration, diagram or three-dimensional visualization should communicate information that text alone cannot communicate as effectively.

If media exists solely because the page feels empty,

it should be removed.

The objective of media is not visual richness.

The objective is understanding through observation.

Users should learn by seeing.

⸻

Media Philosophy

Media should reveal.

Not decorate.

Every piece of media should advance the narrative,

explain a concept,

demonstrate craftsmanship,

or create emotional understanding.

Static images communicate moments.

Video communicates transformation.

Animation communicates process.

Interactive media communicates understanding.

Choose the medium according to the information being communicated.

Self Review

Before implementation verify:

✓ Every media asset has a purpose

✓ Appropriate medium selected

✓ Authentic photography

✓ Cinematic video where transformation matters

✓ Animation explains process

✓ Diagrams simplify complexity

✓ Responsive behavior preserved

✓ Strong performance

✓ Consistent visual language

✓ Media strengthens narrative

If every image, video and animation were replaced by empty placeholders, it should be immediately obvious what information has been lost. If removing a media asset changes nothing about the user’s understanding, that asset did not deserve to exist.
